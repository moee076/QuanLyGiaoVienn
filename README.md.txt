# QuanLyGiaoVien

Ứng dụng Java Swing quản lý giáo viên, kết nối MySQL.

## Chức năng
- Thêm / Sửa / Xóa giáo viên
- Tìm kiếm theo mã, tên, khoa, ngày sinh
- Hiển thị danh sách từ database

## Công nghệ
- Java Swing (NetBeans)
- MySQL
- JDBC

## Hướng dẫn chạy
1. Import project vào NetBeans
2. Tạo database `ql_giaovien`
3. Cấu hình MySQL trong `GiaoVienModify.java`
4. Run file `qlGVFrame.java`
